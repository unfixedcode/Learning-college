# Learning-college

Repositorio académico y portafolio personal de prácticas, ejercicios y proyectos desarrollados durante mi formación en Desarrollo de Software y Aplicaciones.

El portal principal de este repositorio está en `Learning-college/index.html`. Desde ese archivo se accede a los distintos proyectos y prácticas organizados en carpetas internas. La idea de este repositorio es mostrar mi avance real en lógica de programación, pseudocódigo, desarrollo web y construcción de proyectos pequeños e interactivos.

## Portal principal

El archivo principal `index.html` funciona como portal de navegación del repositorio. Desde allí se accede a cuatro áreas principales del proyecto:

- Algoritmos de búsqueda.
- Piedra, papel, periódico y tijera.
- Tricky (Tic Tac Toe).
- Sitio web del restaurante.

Este portal sirve como punto central para visualizar y organizar los distintos trabajos desarrollados durante el proceso formativo.

## Contenido del repositorio

### 1. OtrosAlgoritmos

Carpeta que agrupa proyectos de lógica y JavaScript orientados a práctica de estructuras de decisión, interacción y algoritmos básicos.

#### Busqueda
Proyecto enfocado en búsqueda lineal y búsqueda binaria. Incluye su propio `index.html` en la raíz de la carpeta para probar los ejemplos desde navegador.

#### PPoT
Proyecto interactivo de piedra, papel, periódico y tijera. Incluye `index.html` propio y lógica de juego en JavaScript.

#### Tricky
Proyecto interactivo de Tic Tac Toe, conocido en Colombia como Tricky. Incluye `index.html` propio y lógica del juego con interfaz web.

---

### 2. PSEint

Carpeta con ejercicios iniciales de lógica y pseudocódigo desarrollados en PSeInt.

Estos archivos están pensados para descargarse y abrirse manualmente con el programa PSeInt, ya que no se ejecutan directamente desde navegador.

---

### 3. Web/PrimerCiclo/PaginaCSS-JS

Proyecto de sitio web desarrollado con HTML, CSS y JavaScript, orientado al contexto de un restaurante.

#### Estructura interna
- `.vscode/`: archivos de configuración generados por el entorno de desarrollo.
- `css/`: contiene la hoja de estilos principal `style.css`.
- `docs/`: documentos PDF relacionados con el proyecto.
- `img/`: recursos gráficos e imágenes del sitio.
- `sitio/`: páginas secundarias del proyecto.
- `videos/`: recursos audiovisuales del proyecto.
- `index.html`: página principal del sitio web.

#### Páginas importantes dentro de `sitio/`
- `entregables.html`: incluye material de entrega, como evidencia de despliegue y documento del proyecto.
- `reservas.html`: formulario y lógica visual para registro y reservas de mesas.
- `menu_postres.html`: sección complementaria del menú.
- `politicas_privacidad.html`: página informativa de privacidad.

#### Estado actual
El sitio ya permite navegación, visualización de menús, reservas y estructura de acceso de usuarios. A futuro se planea complementar la lógica con una base de datos estática o simulada para distinguir usuarios registrados, clientes y administrador.

## Repository contents

### 1. OtrosAlgoritmos

Folder that groups logic and JavaScript projects focused on decision structures, interaction, and basic algorithms.

#### Busqueda
Project focused on linear search and binary search. It includes its own `index.html` file at the root of the folder.

#### PPoT
Interactive rock, paper, newspaper, scissors project. Includes its own `index.html` and JavaScript game logic.

#### Tricky
Interactive Tic Tac Toe project, known in Colombia as Tricky. Includes its own `index.html` and game logic.

---

### 2. PSEint

Folder containing early logic and pseudocode exercises developed in PSeInt.

These files are intended to be downloaded and opened manually with the PSeInt software, since they are not meant to run directly in the browser.

---

### 3. Web/PrimerCiclo/PaginaCSS-JS

Website project developed with HTML, CSS, and JavaScript, oriented to a restaurant scenario.

#### Internal structure
- `.vscode/`: configuration files generated by the development environment.
- `css/`: contains the main stylesheet `style.css`.
- `docs/`: PDF documents related to the project.
- `img/`: images and graphic resources.
- `sitio/`: secondary pages of the project.
- `videos/`: audiovisual resources.
- `index.html`: main page of the website.

#### Important pages inside `sitio/`
- `entregables.html`: delivery materials, including deployment evidence and project document.
- `reservas.html`: form and visual logic for user registration and table reservations.
- `menu_postres.html`: complementary dessert menu section.
- `politicas_privacidad.html`: privacy information page.

#### Current status
The website already allows navigation, menu visualization, reservations, and user access structure. In the future, it is planned to complement the logic with a static or simulated database to distinguish registered users, clients, and administrator roles.
