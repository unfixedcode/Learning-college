# Learning-college

Este repositorio reúne mis prácticas, ejercicios y proyectos realizados durante mi formación en la Tecnología en Desarrollo de Software y Aplicaciones. Su objetivo es mostrar mi progreso real en lógica de programación, estructura de código y buenas prácticas, desde los primeros algoritmos hasta ejercicios más complejos.

Aquí podrás encontrar carpetas organizadas por tecnología o temática (por ejemplo, PSeInt, JavaScript, etc.), donde cada una contiene ejercicios específicos con su propio contexto y README. La idea es que cualquier persona pueda ver cómo voy mejorando en claridad, orden y pulcritud en el código a lo largo del tiempo.

## ¿Qué puedes encontrar aquí?

- Ejercicios básicos y avanzados de algoritmos.

Cada carpeta incluye una descripción de lo que contiene y cómo ejecutar los ejemplos, para que se puedan revisar y probar fácilmente.
